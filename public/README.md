# OPAC System (Sulong-Dunong)

## 1. Introduction

The **Online Public Access Catalog (OPAC) System** is a software application designed to facilitate efficient access to library resources. This system enables users to search, retrieve, and manage bibliographic records through a structured and user-friendly interface.

## 2. Objectives

The primary objectives of this project are:

* To provide an accessible platform for searching library materials
* To improve the organization and management of catalog records
* To enhance user experience in accessing library information

## 3. Features

* Comprehensive search functionality
* Structured display of bibliographic information
* Administrative tools for catalog management
* Organized and scalable system architecture

## 5. Installation Procedure

To set up the project locally, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/alyencs/Sulong-Dunong.git
   ```

2. Navigate to the project directory:

   ```bash
   cd Sulong-Dunong
   ```

3. Open the project using a code editor such as Visual Studio Code.

4. Configure the database and environment variables as required.

5. Run the application using a local development server.

## 6. Usage

Once the system is running:

* Users may search for available resources through the search interface
* Administrators may manage catalog entries through the administrative module
